One-time setup required:
1. Open the Google Sheet
2. Click Extensions → Apps Script
3. Delete any existing code and paste the Apps Script code (provided separately)
4. Click Deploy → New Deployment → Web App
5. Set "Execute as" = Me, "Who has access" = Anyone
6. Copy the deployment URL and paste it in the file where it says APPS_SCRIPT_URL = ''
7. Save and reload this file
